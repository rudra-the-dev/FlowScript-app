
# FlowScript

A programming language that reads like plain English and automates your Android phone.

## What it does
- Write automation scripts in plain English
- Control any app on your phone
- Send WhatsApp messages, open apps, scroll, tap — all from simple code

## Example
send whatsapp to "Mom" saying "On my way home"
open "YouTube" and search "lo-fi beats"
repeat 3 times open "Instagram"

## Status
🚧 Under active development

## Tech Stack
- Python + Kivy (Android app)
- Flask (Vision backend)
- OpenRouter AI (Screen understanding)
- Android Accessibility Service (UI automation)
